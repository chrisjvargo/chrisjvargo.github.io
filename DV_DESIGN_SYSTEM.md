# DV Design System

The DV section uses static HTML, local CSS, system fonts, no trackers, no cookies, and no runtime chart library.

The visual style is a restrained evidence brief: warm off-white background, near-black text, neutral rules, and a copper accent that does not encode sex or gender. Charts are deterministic SVG/HTML generated from the public release.

The root academic site remains otherwise unchanged except for the `DV Research` navigation link.
