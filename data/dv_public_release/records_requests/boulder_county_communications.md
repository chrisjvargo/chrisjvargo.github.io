# Boulder County Communications Records Request

Custodian: Boulder County Communications.

Access regime to verify before sending: CCJRA for CAD/911 records; CORA for administrative code tables and policy records.

Request deidentified CAD records for potentially domestic-related calls dispatched to Boulder County agencies, including initial and final call type, timestamps, location geography sufficient for jurisdiction without exact address, responding agency/unit, disposition, transfer/assist information, and stable call/event keys. Request CAD code dictionaries, dispatch protocols, domestic/family/civil-standby coding guidance, and system migration/change logs.

Request metadata for 911 recordings for the analytic population and a stratified sample design before any audio production.
