# Colorado Judicial Branch Compiled Data Request

Custodian: Colorado Judicial Branch.

Access regime: compiled-data procedure under Chief Justice Directive 05-01, section 4.40, using Addendum A unless Judicial Branch directs otherwise.

Request deidentified count-level Twentieth Judicial District data with anonymous case and defendant keys, filing and disposition dates, referring or arresting agency where available, original/amended/final charges, statutory offense codes, domestic-violence findings/designations, protection-order entries, service/actual-notice indicators where releasable, pleas, verdicts, dismissals and reasons, sentence and treatment conditions, probation outcomes, reopened cases, and sealed/suppressed indicators sufficient to evaluate missingness without exposing sealed records.

Request CSV/TSV/Parquet, data dictionary, code tables, fee estimate before work, and rolling production if feasible.
