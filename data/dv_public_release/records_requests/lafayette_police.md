# Lafayette Police Department Records Request

Custodian: City of Lafayette Police Department.

Access regime to verify before sending: CCJRA for criminal-justice records; CORA for administrative policy, training, and data-system records.

Request deidentified structured CAD/RMS/enforcement/referral records for domestic-related, family, civil-standby, custody, property-retrieval, harassment, disturbance, welfare-check, assault, and protection-order calls. Include no-arrest and report-only events. Request code tables, policy/training versions, dispatch protocols, body-camera policies, and system migration logs.

Request metadata first for narratives, 911 audio, photographs, and body-camera/dash-camera records, followed by a statistically stratified sample.
