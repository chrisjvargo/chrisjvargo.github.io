# Boulder County Jail Records Request

Custodian: Boulder County Jail / Boulder County Sheriff's Office.

Access regime to verify before sending: CCJRA for booking records; CORA for administrative data dictionaries and system records.

Request deidentified booking records for domestic-violence-related arrests and linked non-DV comparison arrests, including booking key, arresting agency, booking date/time, release date/time, arrest charges, holds, bond/protection-order indicators where releasable, and stable deidentified person/case keys. Request data dictionaries, charge-code tables, and migration/change logs.
